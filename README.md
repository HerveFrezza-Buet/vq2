# vq2
C++ generic programming for vector quantization

# Installation

First, get the files.

``` 
git clone https://github.com/HerveFrezza-Buet/vq2
``` 


Then, install can be done as follows with cmake.

``` 
cd <your_path_here>/vq2
mkdir -p build
cd build
cmake .. -DCMAKE_INSTALL_PREFIX=/usr
sudo make install
```
